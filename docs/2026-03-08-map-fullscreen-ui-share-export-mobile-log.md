# 2026-03-08 Fullscreen Map UI / Share / Export / Mobile Controls

## 문제 -> 해결
- 문제: 임시 변경 상태가 헤더 레이아웃을 깨뜨리고 MapControls 버튼이 헤더를 벗어남.
  해결: 임시 변경 라벨/버튼을 구조맵 영역 우측 상단으로 분리하고 MapControls는 헤더 고정 유지.

- 문제: 임시 변경 라벨이 버튼처럼 보이고 완료/발행 버튼이 존재감이 약함.
  해결: 라벨을 텍스트 성격으로 분리하고 완료/발행 버튼 크기/배경 강화.

- 문제: Ctrl/Cmd+F가 페이지 내 검색 UI와 연동되지 않음.
  해결: 키다운 핸들러로 헤더 검색 UI를 열고 포커스 처리.

- 문제: 검색창이 너무 눈에 띄지 않음.
  해결: 검색창 배경을 어두운 톤으로, 텍스트는 흰색으로 조정 후 밝기 재조정.

- 문제: 구조맵 이미지/PDF 저장 기능 부재.
  해결: Mind Elixir exportPng를 노출하고 헤더 더보기 메뉴에 PNG 저장 추가.

- 문제: 마크다운 복사 기능이 필요하며 컨텍스트 메뉴 확장 필요.
  해결: contextMenu.extend로 “마크다운 복사”/“펼쳐진 노드만 복사” 추가, 노트 포함, 성공/실패 토스트 추가.

- 문제: 컨텍스트 메뉴 기본 항목(포커스 모드 등) 사라짐.
  해결: contextMenu에 focus/link 옵션을 함께 지정해 기본 메뉴 복원.

- 문제: 공유 기능 부재.
  해결: 공유 토큰 기반 링크 생성/해제 API, 공유 페이지(읽기 전용) 추가, 공유 모달 UI 제공.

- 문제: 읽기 전용 페이지에도 핵심 컨트롤 필요.
  해결: 읽기 전용 페이지 전용 컨트롤 레이아웃 구성 및 헤더에 배치.

- 문제: Mind Elixir 기본 툴바의 전체보기 버튼이 커스텀 기능과 충돌.
  해결: 툴바 첫 아이콘 숨김(추가로 툴바 자체는 비활성화) 및 헤더에 가운데/확대/축소 아이콘 제공.

- 문제: 헤더 내 아이콘 라벨 브레이크포인트가 너무 이른 시점에 사라짐.
  해결: 아이콘-only 전환 기준을 680px -> 760px로 조정.

- 문제: 모바일 헤더가 잘림/혼잡, 검색창 펼침 시 레이아웃 붕괴.
  해결: 모바일은 헤더 2줄 유지, 하단 줄 높이 축소, 검색창 z-index 상승 및 검정 배경 적용.

- 문제: 모바일 컨트롤이 잘려 보임.
  해결: 모바일 전용 세로 아이콘 바(오른쪽) 구성, 맵 조작/테마 드롭다운 분리.

- 문제: 상단 타이틀 바 정렬 및 닫기 버튼 위치 혼란.
  해결: 타이틀 바는 모바일 좌측/데스크탑 중앙 정렬, 닫기 버튼은 항상 우측 끝.

- 문제: LeftPanel 닫기 버튼 및 타이틀 중복.
  해결: 닫기 버튼 제거, 정보 탭에서만 타이틀 표시(모바일/데스크탑 공통).
