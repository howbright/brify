# Global SEO Checklist for "AI mind map conversion" (KO/EN/FR)

## 1) Technical indexing (once)
- Confirm sitemap: `https://www.brify.app/sitemap.xml`
- Confirm robots: `https://www.brify.app/robots.txt`
- Confirm canonical + hreflang on:
  - `/ko`, `/en`, `/fr`
  - `/ko/ai-mindmap-convert`, `/en/ai-mindmap-convert`, `/fr/ai-mindmap-convert`
  - `/ko/mind-map`, `/en/mind-map`, `/fr/mind-map`
- Confirm `x-default` points to English versions.

## 2) Search Console submission (Google + Naver)
- Submit sitemap once.
- Request indexing for priority URLs (see `seo-priority-urls.md`).
- Re-check indexing status in 3-7 days.

## 3) Keyword focus by locale
- Korean primary: `AI 마인드맵 변환`
- English primary: `AI mind map conversion`
- French primary: `conversion IA en carte mentale`

## 4) On-page copy rules (all locales)
- Include primary keyword in:
  - page title
  - meta description
  - first paragraph (lead)
- Keep intent words in title:
  - convert / transform / 구조화 / structured
- Mention benefit in description:
  - free start / fast conversion / language support

## 5) Internal links
- Add links from home + related landing + blog posts.
- Use anchor text with the target keyword (not generic "click here").

## 6) Content publishing cadence (first 4 weeks)
- Week 1-2: publish 3-4 posts per locale (KO first, then EN/FR).
- Week 3-4: publish 2-3 additional posts targeting long-tail keywords.
- Each post: 1 main keyword + 1-2 secondary keywords + 1 landing link.

## 7) CTR optimization loop
- Every 2 weeks:
  - check impressions / CTR by query in Search Console
  - adjust title + description for low CTR pages
  - keep keyword, improve value proposition wording
